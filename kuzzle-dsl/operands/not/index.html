        not

        Since Kuzzle v1.0.0

        The not filter reverts a filter result.

        Given the following documents:

        {
  firstName: 'Grace',
  lastName: 'Hopper',
  city: 'NYC',
  hobby: 'computer'
},
{
  firstName: 'Ada',
  lastName: 'Lovelace',
  city: 'London',
  hobby: 'computer'
}

        The following filter validates the first document:

        {
  not: {
    equals: {
      city: 'London'
    }
  }
}
