# Schema Structure

The validation schema must follow the following structure to be valid:

```json
{
  "myIndex": {
    "myCollection": {
      "strict": true,
      "fields": {
        "fieldName": {
            "mandatory": true,
          "type": "string",
          "defaultValue": "a default value",
          "multivalued": {
            "value": true,
            "minCount": 1,
            "maxCount": 5
          },
          "typeOptions": {
            "length": {
              "min": 2,
              "max": 12
            }
          }
        },
        "anotherFieldName": {
          "..."
        },
        "myObjectField": {
          "type": "object",
          "..."
        },
        "myObjectField/mySubField": {
          "..."
        }
      },
      "validators": [
          "..."
      ]
    },
    "..."
  },
  "..."
}
```
