# getJwt

```js
var jwt = kuzzle.getJwt();
```

```java
String jwt = kuzzle.getJwt();
```

```php
<?php
use \Kuzzle\Kuzzle;

$kuzzle = new Kuzzle('localhost');
// ...
$token = $kuzzle->getJwt();
```

Get internal Json Web Token used to request kuzzle.

---

## Return Value

Returns the stored JWT as a string value.
