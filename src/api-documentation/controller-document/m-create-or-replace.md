# mCreateOrReplace


<blockquote class="js">
<p>
**URL:** `http://kuzzle:7512/<index>/<collection>/_mCreateOrReplace[?refresh=wait_for]`  
**Method:** `PUT`  
**Body:**
</p>
</blockquote>


```js
{
  "documents": [
    {
      "_id": "<documentId>", // Mandatory
      "body": {
        "document": "body",

    {
      "_id": "<anotherDocumentId>" // Mandatory
      "body": {
        "document": "body",
        ...
      }
    },

}
```

```json
{
  "index": "<index>",
  "collection": "<collection>",
  "controller": "document",
  "action": "mCreateOrReplace",
  "refresh": "wait_for",
  "body": {
    "documents": [

        "_id": "<documentId>",
```
