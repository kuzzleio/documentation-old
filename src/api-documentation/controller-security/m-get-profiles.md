# mGetProfiles



<blockquote class="js">
<p>
**URL:** `http://kuzzle:7512/profiles/_mGet`  
**Method:** `POST`  
**Body:**
</p>
</blockquote>


```js
{
  // ids must be an array of profile ids
  "ids": ["firstProfileId", "secondProfileId"]
}
```

<blockquote class="json">
<p>
**Query**
</p>
</blockquote>

```json
{
  "controller": "security",
  "action": "mGetProfiles",
  "body": {
    "ids": ["firstProfileId", "secondProfileId"]
  }
}
```
