          ms (memoryStorage)

          Events triggered when a request is sent to the memoryStorage controller.

          EventTypeDescriptionPayload
          ms:after<Action>PipeAll actions in memoryStorage controller trigger an event after executingType: Request
          ms:before<Action>PipeAll actions in memoryStorage controller trigger an event before executingType: Request
          ms:error<Action>PipeWhen an error occurred during an action in ms controller, an event error is triggeredType: Request
