          bulk

          Since Kuzzle v1.0.0

          Events triggered when a request is treated in the bulk controller

          bulk:afterImport

          Since Kuzzle v1.0.0

          Event type: Pipe

          Payload: a Request object

          Triggered after a successful execution of a bulk import request.

          bulk:beforeImport

          Since Kuzzle v1.0.0

          Event type: Pipe

          Payload: a Request object

          Triggered before a bulk import request is executed.

          bulk:errorImport

          Since Kuzzle v1.0.0

          Event type: Pipe

          Payload: a Request object

          Triggered after an execution of a bulk import request failed.
